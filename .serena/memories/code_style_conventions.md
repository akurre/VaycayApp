# Code Style and Conventions

## TypeScript Code
- ESLint 9 with TypeScript ESLint 8
- Prettier 3.4+ for formatting
- Strict TypeScript configuration
- Code-first GraphQL with Nexus

## Linting & Formatting
- **Lint**: `make lint` or `make lint-fix`
- **Format**: `make format` or `make format-check`
- **Type Check**: `make type-check`

## Git Hooks
- Husky 9.0.11 for git hooks
- lint-staged for pre-commit checks

## Testing
- Jest for testing
- Run with `make test` for coverage reports
